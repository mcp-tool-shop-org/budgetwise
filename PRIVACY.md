# Privacy Policy for NextLedger

**Last Updated: February 4, 2026**

## Overview

NextLedger is a personal finance application designed with privacy as a core principle. Your financial data stays on your device.

## Data Collection

**NextLedger does not collect, transmit, or store any personal data on external servers.**

### What stays on your device:
- All financial transactions
- Account information
- Budget allocations
- Application settings

### What we don't collect:
- Personal identification information
- Financial account credentials
- Usage analytics
- Location data
- Any data transmitted to external servers

## Data Storage

All data is stored locally on your Windows device using SQLite database. The data remains under your complete control and is never transmitted to any external service.

## Third-Party Services

NextLedger does not integrate with any third-party analytics, advertising, or tracking services.

## Optional Features

### XRPL Integration (Optional)
If you choose to use the optional XRP Ledger integration features, transactions you initiate will interact with the public XRP Ledger blockchain. This is entirely opt-in and requires explicit user action.

## Data Deletion

You can delete all your data at any time by:
1. Uninstalling the application
2. Deleting the application data folder

## Children's Privacy

NextLedger does not knowingly collect information from children under 13.

## Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted to this page with an updated revision date.
